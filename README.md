# Instaclone

## Instaclone-backend
노마드코더의 [풀스택]인스타그램 클론코딩 강의를 들으며 진행한 프로젝트입니다.

## 기술 스택
- prisma
- apollo
- graphql
- javascript
- aws
- pubsub
